## 控制层模块概述

隐私计算互联互通控制层模块包含流程调度和算法容器管理两部分内容，流程调度约定了互联互通平台基于南北向、东西向的作业调度控制以及任务调度控制接口，算法容器管理明确了平台对于算法容器的管理机制，包含组件注册与发现以及容器管理与加载。架构图如下：
